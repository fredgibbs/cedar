Methodology

All data comes from UNM MyReports, specifically the Department Enrollment Status Report. Some filtering and aggregating makes this report more readable, namely:

  • filtering out courses like independent studies, dissertation hours, etc.
  • compressing cross-listed courses into a single row to display both department and crosslist enrollments
  • filtering out summer courses from the workload matrix

Job titles are taken from HRReports, but these are not fully normalized. “–” as a value means there is no match in the A&S HR Report (ie their contract comes from another College).
